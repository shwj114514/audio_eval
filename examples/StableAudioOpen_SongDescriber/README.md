# Stable Audio Open + Song Describer TTM example

This example evaluates one 47-second music sample generated by the open-source
`stabilityai/stable-audio-open-1.0` model from the Song Describer no-singing
evaluation set (`caption_id=10`). It intentionally has no `ref_path`; CLAP
evaluates text-audio alignment for this single generated sample. Distribution
metrics such as FD and KL require a complete evaluation set.

Generation settings: 44.1 kHz, 100 steps, CFG 7.0, `dpmpp-3m-sde`, sigma range 0.3-500, seed 10.

```bash
bash examples/Stable_Audio_Open_SongDescriber/run.sh
```
