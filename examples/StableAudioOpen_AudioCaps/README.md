# Stable Audio Open + AudioCaps TTA example

This example evaluates one 10-second general-audio sample generated by the
open-source `stabilityai/stable-audio-open-1.0` model from an AudioCaps test
prompt (`audiocap_id=103548`). It intentionally has no `ref_path`; CLAP
evaluates text-audio alignment for this single generated sample. Distribution
metrics such as FD and KL require a complete evaluation set.

Generation settings: 44.1 kHz, 100 steps, CFG 7.0,
`dpmpp-3m-sde`, sigma range 0.3-500, seed 2.

```bash
bash examples/Stable_Audio_Open_AudioCaps/run.sh
```
